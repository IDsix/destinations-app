# TripX Destinations

A Next.js travel booking interface that displays and filters destinations fetched from the TripX API.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **SCSS Modules**
- **lucide-react** for icons

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
### Setup Environment Variables

Duplicate `.env.example` to `.env.local` and fill in the needed values:

`DESTINATIONS_API_URL` - TripX destinations endpoint  
`LOGIN_API_URL` - TripX login endpoint 

### Running the destinations app
First install dependencies under project root
```bash
npm install
```
If you want to run the development server use
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
